## 三种模式，总有一种适合你

很多人一开始觉得 Manico 就是个简单的启动器，但实际上它支持三种模式，覆盖了大部分使用场景：

**Dock 模式**：自动映射你的 Dock 栏，Option + 数字键直接对应 Dock 上第几个图标。零配置，装完即用，最适合新手。

**切换器模式**：映射当前正在运行的 App，类似系统自带 Command + Tab，但快很多，而且可以直接隐藏 App。如果你不喜欢固定 Dock 栏，这个模式更灵活。

**自定义模式**：完全自己说了算，把最常用的 App 拖进来，想绑哪个键绑哪个键。重度效率用户最爱这个。

我用的是自定义模式，把浏览器、终端、编辑器、Slack、备忘录各绑一个键，日常工作基本不用鼠标了。
